# App Store Connect URLs

Use these public URLs when submitting haixoai to the App Store.

| Field | URL |
|---|---|
| **Marketing URL** | https://haixoai.com/ |
| **Support URL** | https://haixoai.com/support.html |
| **Privacy Policy URL** | https://haixoai.com/privacy.html |

## Notes

- The marketing URL is the main landing page (`index.html`).
- The privacy policy HTML page is derived from `PRIVACY_POLICY.md` in this folder. When you update the policy, edit the markdown file first, then sync the changes to `privacy.html`.
- Apple requires the privacy policy URL to be publicly accessible without login.
- Ensure App Store Connect **App Privacy** answers match what is described in the privacy policy.
